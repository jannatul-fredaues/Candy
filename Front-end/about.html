About Candy

Candy started as a question: what would it feel like if a voice assistant actually felt like a companion, not a command line you talk to? This is the project that came out of chasing that answer.

Our Mission

Voice interaction that feels alive

Most voice assistants are fast but forgettable — they answer and reset. Candy is built around the opposite idea: a companion that listens, remembers, and responds with a personality of its own, expressed through real-time speech recognition, natural language understanding, and a face that actually reacts to you.

The goal isn't just accurate transcription. It's closing the gap between "the software understood my words" and "the software understood me."

The Journey

From sketch to orb

Candy is a final year project, built end to end — from research and architecture to the floating, breathing interface you see on the homepage.

Phase 01
Research & Concept
Studied existing voice assistants, identified the gap between functional and felt — and sketched the idea of an orb-based companion instead of a chat box.
Phase 02
Core Voice Engine
Built the speech recognition and NLP pipeline, tuned for low-latency, real-time response.
Phase 03
Memory & Personality
Layered in context retention so Candy remembers preferences across a conversation, plus the expressive orb face and animation system.
Phase 04
Polish & Presentation
Refined multilingual support, performance, and the interface — landing on the version you're using today.
What We Believe

Principles behind the build

💜
Personality First
An assistant should feel like someone, not something. Every detail — voice, face, timing — is designed around that.
⚡
Speed Is Respect
A laggy assistant breaks trust. Sub-second response time was a non-negotiable design constraint, not an afterthought.
🧠
Context Matters
Conversations aren't one-off queries. Candy holds onto context so it doesn't make you repeat yourself.
🌐
Built for Everyone
Language shouldn't be a barrier to a good assistant — multilingual support was core from day one.
